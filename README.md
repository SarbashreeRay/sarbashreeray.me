# Sarbashree Ray Portfolio Website

A premium, modern, responsive portfolio website for **Sarbashree Ray** (Senior Data Scientist, MLOps Engineer, and GenAI Architect). Designed with high-end glassmorphism elements, interactive visual widgets, and a fully custom dark/light theme switching capability.

## Features

- **No-Photo Visual Design**: The typical photo section is replaced with a premium, animated, and interactive SVG Neural Network / AI Globe illustration representing AI/ML operations.
- **Micro-Interactions**: The SVG globe wiggles and wisp-translates its nodes based on your mouse hover, and the network connection lines glow dynamically.
- **Theme Toggling**: Dark and Light themes are fully styled and managed seamlessly, with preference caching inside local storage.
- **Responsive Navigation**: Mobile-responsive layout, hamburger drop-down menu, and dynamic scroll active-tracking via `IntersectionObserver`.
- **Interactive Experience & Skills**: Tabbed skill views, progress transitions, and vertical timeline elements.
- **Optimized for SEO**: Structured headers, meta tags, and structured HTML semantic elements.

## Tech Stack

- **Markup**: Semantic HTML5
- **Styles**: Custom HSL-tailored vanilla CSS3 (with CSS variables and custom `@keyframes` animations)
- **Logic**: Vanilla ES6 JavaScript
- **Icons**: FontAwesome v6.4

## Project Structure

## Local Development

Simply open `index.html` in any modern web browser, or launch it with a live server extension (like VS Code Live Server).
